# 가이드윤의 바른 생활 Community service

## Front-end 소개

- 규칙적인 생활을 위한 일상 공유 커뮤니티 서비스 '가이드윤의 바른 생활'입니다.
- FE의 기본적인 학습을 위해 프레임워크 없이 구현했습니다.
- 개발은 초기 프로젝트 화면부터, 기능, 백엔드 연결까지 LLM을 이용하여 구현했습니다.

### 개발 인원 및 기간

- 개발기간 : 2025-09-15 ~ 2025-12-07
- 개발 인원 : 프론트엔드/백엔드/인프라 1명 (본인)

### 사용 기술 및 tools

- Express.js

### Back-end

- <a href="https://github.com/brixxt27/jayoon-community-backend">Back-end Github</a>

## 프로젝트 후기

저는 평소에 BE만을 위주로 개발해서 FE의 F자도 모르는 상황이었습프다.니다. 하지만 이번 프로젝트는 처음으로 FE 구조를 고민하고 이를 기반으로 LLM을 통해 개발을 했습니다. 이번 프로젝트는 인프라와 백엔드를 중심으로 고민했고, FE는 여기에서 어떻게 BE와 인프라에서 효과적인 성능을 낼 수 있을까를 생각했습해다.

# QA check list

## Bug report

---

### 헤더

- [ ] 사용하다보면 내 이미지가 나오지 않고, 기본 이미지가 로드된다.
  - 브라우저 캐싱 문제?
  - session의 user 객체 내 profileImageUrl 업데이트 되는지 추적하기

### 게시물 조회

- [ ] 내가 작성한 게시물인데 기본 프로필 이미지가 나옴
  - Session storage에는 user:"{"userId":7,"email":"jayoon3@gmail.com","nickname":"jayoon5","profileImageUrl":"https://jayoon-bucket.s3.ap-northeast-2.amazonaws.com/public/image/profile/image-1765239143391-473968485.jpg"}"가 정상적으로 저장 되어있습니다.

### 회원 정보 수정 페이지

- [x] 회원 정보 수정 시 프로필 이미지가 클릭이 되지 않음
  - 변경 후 헤더를 재로드 하지 않아서 발생한 문제

### 댓글

- [ ] 기존 댓글이 존재할 때 댓글수가 추가 되지 않고, 0부터 오른다.

## UX 개선 사항

---

### 공통

- [ ] 게시글 작성 후 뒤로가기 누르면 다시 작성 페이지로 넘어간다. 이 부분이 괜찮은지 아닌지 고민해보기
- [ ] 이미지 로드 늦음

### 로그인 하지 않았을 때

- [ ] 댓글 생성
- [ ] 좋아요 생성 또는 삭제
- [ ] 글쓰기 시 아무런 메시지 없이 로그인 페이지로 넘어감

### 로그인 했을 때

- [ ] 게시물 작성 시 alert
- [ ] 게시물 삭제 시 alert
- [ ] 비밀번호 수정 완료 시 토스트 뜨고 리다이렉션 되지 않음
- [ ] 닉네임만 수정하면 프로필 이미지 클릭이 안 됨. 프로필 이미지 수정과는 관련이 없음
